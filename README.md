# Banner Report

A LaTeX class for a two column report with a fancy title page.



## Title page

Call the `	\titlepage{banner.pdf}` command at the start of your document, with `banner.pdf` as the title of the banner file. The file should be a 3.55 in Ã— 9.6 in PDF.

You can fill in the other areas on the page with the following commands 

```latex
\title{Banner Report}
\subtitle{A Stylish \LaTeX Class}
\info{Sep 2019 \\ v0.8}
\author{ {\small prepared by} \\ Arya Daroui}
```
